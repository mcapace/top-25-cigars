# Cigar Aficionado Brand Colors

This document outlines the official brand colors used in the Top 25 Cigars project.

## Primary Colors

- **Cream Background**: `#f2ece6` - Main background color
- **Dark Brown Text**: `#4a1b01` - Primary text color (very dark brown/almost black)
- **Dark Brown Accent**: `#58210d` - Secondary dark brown for accents

## CSS Variables

The following CSS variables are defined in `src/app/globals.css`:

```css
--color-cream: #f2ece6;          /* Light cream background */
--color-warmWhite: #FFFDF9;      /* Warm white for highlights */
--color-charcoal: #4a1b01;       /* Dark brown for text */
--color-darkBrown: #58210d;      /* Medium dark brown for accents */
--color-amber: #C4501C;          /* Amber for highlights/accents */
--color-gold: #D4A84B;           /* Gold for special elements */
--color-stone: #78716C;          /* Stone gray for secondary text */
```

## Usage

- **Background**: Use `#f2ece6` for main page backgrounds
- **Primary Text**: Use `#4a1b01` for main headings and body text
- **Accents**: Use `#58210d` for borders, buttons, and secondary elements

## Tailwind Classes

The colors are available as Tailwind utility classes:
- `bg-cream` - Light cream background
- `text-charcoal` - Dark brown text
- `text-darkBrown` - Medium dark brown text
- `border-darkBrown` - Dark brown borders
- etc.

## Notes

All brand colors have been integrated into the global styles and are ready for use throughout the application components.

# Cigars Directory

Place cigar product images here, one for each of the Top 25 cigars.

## Naming Convention

Name files according to their rank:
- `1.jpg` - Rank #1 (Cigar of the Year)
- `2.jpg` - Rank #2
- `3.jpg` - Rank #3
- ... and so on through `25.jpg`

## Image Specifications

- Format: JPG or PNG
- Recommended size: 1200x1200px or larger
- Aspect ratio: 1:1 (square) or vertical
- Quality: High resolution for web display

## Notes

- Images should be optimized for web (compressed but high quality)
- The image paths are configured in `src/lib/data/cigars.server.ts`
- Update the `imageUrl` field in the cigar data to match your file names

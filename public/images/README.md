# Images Directory

This directory contains all image assets for the Cigar Aficionado Top 25 project.

## Structure

- `logos/` - Cigar Aficionado brand logos
- `cigars/` - Cigar product images (one per rank, named 1.jpg through 25.jpg)

## Image Naming Convention

- Cigar images: `1.jpg`, `2.jpg`, `3.jpg`, etc. (matching rank numbers)
- Logo files: Use descriptive names like `cigar_aficionado_logo.png`
